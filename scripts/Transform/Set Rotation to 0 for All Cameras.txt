Beb.Tools - Zero All Camera Rotations Script
============================================

Description:
This script sets the rotation of all camera
objects in the scene to (0, 0, 0) degrees
on the X, Y, and Z axes.

Usage:
1. Add camera objects to your scene.
2. Open the N-Panel (press N).
3. Go to the "Beb.Tools" tab.
4. Select this script from the list.
5. Click "Run" to run the script.

What It Does:
- Iterates through all camera objects in the scene.
- Sets each camera objectâ€™s rotation to (0, 0, 0).

Notes:
- Only affects camera objects.
- Ignores non-camera objects (e.g., lights, meshes).
- Only changes rotation; location and scale unchanged.
- Use Undo to revert if needed.

Output:
- See console message:
  "All camera object rotations set to (0, 0, 0)!"
