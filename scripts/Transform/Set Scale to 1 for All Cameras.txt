Beb.Tools - Set Scale to 1 for Cameras Script
=============================================

Description:
This script sets the scale of all camera
objects in the scene to (1, 1, 1) on
the X, Y, and Z axes.

Usage:
1. Add camera objects to your scene.
2. Open the N-Panel (press N).
3. Go to the "Beb.Tools" tab.
4. Select this script from the list.
5. Click "Run" to run the script.

What It Does:
- Iterates through all camera objects in the scene.
- Sets each camera objectâ€™s scale to (1, 1, 1).

Notes:
- Only affects camera objects.
- Ignores non-camera objects (e.g., lights, meshes).
- Only changes scale; location and rotation unchanged.
- Use Undo to revert if needed.

Output:
- See console message:
  "All camera object scales set to (1, 1, 1)!"
